# ⚽ Football Predictor

Chatbot de terminal en Python que genera pronósticos deportivos completos usando datos reales de la API de fútbol y la inteligencia de Claude de Anthropic.

## 🚀 Funcionalidades

- Listado de partidos de fútbol disponibles hoy
- Búsqueda de partidos por nombre de equipo (en español)
- Convocatoria real de ambos equipos vía API-Football
- Últimos 5 partidos de cada equipo
- Pronóstico completo con resultado probable, goleadores, corners y mercados de apuesta
- Respuestas en español

## 🛠️ Tecnologías

- Python 3
- [Anthropic API](https://www.anthropic.com)
- [API-Football](https://www.api-football.com)
- python-dotenv
- requests

## ⚙️ Instalación

1. Clona el repositorio
2. Instala las dependencias: `pip install anthropic python-dotenv requests`
3. Crea un archivo `.env` con tus claves
4. Ejecuta: `python main.py`

## ⚠️ Aviso

Los pronósticos son orientativos y se generan con datos reales pero no garantizan resultados. Úsalos con responsabilidad.
